### New Post
```
rake post
```
OR
```
rake post['my new post']
```

### Generate site
```
rake generate
```

### Deploy to github pages
```
rake deploy
```
Make sure you are on **source** branch

### Local server
```
rake watch
```
OR
```
jekyll serve
```
